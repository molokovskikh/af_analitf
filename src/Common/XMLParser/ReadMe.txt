      
		Описание модулей для работы с XML.

U_ESGXMLError: содержит класс исключений, вызываемых при ошибках в XML.

U_XMLElementContents: содержит описание структуры данных для работы с содержимым
 	DTD-описания. На данном этапе не потребуется.


U_TSGDTDElement: содержит структуру для работы с DTD-описанием элемента.
	На данном этапе не потребуется.

U_TSGDTD: содержит структуру для работы с DTD-описанием.
	На данном этапе не потребуется.

U_TSGXMLAttribute: содержит структуру для работы с XML-аттрибутом.
	Основные поля: AttName, AttType, Value.
	Функции IsXXX возвращают True, если аттрибут является типом XXX.
	Процедуры AsXXX возвращают аттрибут как значение типа XXX при ошибки вызывая исключение.

U_XMLElementStructs: содержит структуры XML-элементов.
	Основная структура : TSGXMLElement. Также есть TSGXMLPCData - секции PCDATA в документе,
	у которой основное поле Value.
	Основные поля TSGXMLElement : 
		XMLElemName - имя элемента,
		Items	    - потомки элемента, которые могут не быть. Индексированное поле.

U_TSGXMLDocument: содержит класс для работы с XML-документом. Является прямым потомком TSGXMLElement.
	Основное поле RootName - имя корневого элемента, если документ заполняется не с помощью
	TSGXMLParser'а, то это поле обязательно должно быть заполнено вручную. 
	Т.к. он является потомком, то при вызове унаследованных методов отправной точкой служит
	корневой элемент. Например, 
	var XMLDoc : TSGXMLDocument;
	begin
	  XMLDoc := TSGXMLDocument.Create;
	  XMLDoc.RootName := 'Root'; //Присвоили имя корневому элементу
	  XMLDoc.AddElement('Level1');//Добавили к корневому элементу дочерний элемент Level1		
	end;

U_TSGXMLParser: содержит класс, осуществляющий разбор XML-документа.
	Основные процедуры : 
		LoadFromFile,
		LoadFromStream - загружают нужный документ,
		ParseXML(AXMLDocument : TSGXMLDocument) - разбирает документ и 
							  заносит в AXMLDocument.

U_TXMLFrame: содержит визуальный компонент TXMLFrame для отображения XML-документа.
	Запрашивает следующие модули : U_TSGXMLAttribute, U_TXMLElementStructs, U_TSGXMLDocument.
	Дочерние компоненты : 
		tvXML 	      - отображает в дерево структуру документа,
		lbAttributes  - отображает список аттрибутов выделенного в TTreeView элемента.
		mValue	      - отобоажет значения аттрибута или секции PCDATA.
	Основная процедура InputXML(AXMLDocument : TSGXMLDocument). Если требуется очистить
	компонент, то требуется вызвать процедуру InputXML с параметром nil.
